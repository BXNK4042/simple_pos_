# ✅ MVP Checklist: Grocery Store POS System

## 🛒 1. Product Management

- [ ] Add products
- [ ] Edit products
- [ ] Delete products
- [ ] Set product name
- [ ] Set SKU (Stock Keeping Unit)
- [ ] Set barcode (optional)
- [ ] Assign product category
- [ ] Set selling price
- [ ] Set cost price
- [ ] Set initial stock quantity
- [ ] Enable/disable product availability

---

## 📦 2. Inventory Management

- [ ] Add stock
- [ ] Increase stock quantity
- [ ] Decrease stock quantity
- [x] Automatically deduct stock after a sale
- [x] Display current stock levels
- [x] Low-stock alerts

---

## 💵 3. Point of Sale (POS)

- [x] Search products
- [x] Scan barcode
- [x] Add items to cart
- [x] Update item quantity
- [x] Remove items from cart
- [x] Automatically calculate total
- [x] Cancel sale before payment

---

## 💳 4. Payment

- [ ] Support cash payments
- [ ] Support QR code payments
- [ ] Calculate change
- [x] Record payment amount
- [x] Record transaction date and time
- [x] Mark transaction as "Paid"

---

## 🧾 5. Receipt

- [x] Generate receipt number
- [x] Display transaction date and time
- [x] Display purchased items
- [x] Display item quantities
- [ ] Display unit prices
- [x] Display total amount
- [ ] Display payment method
- [ ] Print receipt
- [ ] Export receipt as PDF

---

## 📊 6. Sales Reports

- [ ] Daily sales report
- [ ] Sales report by date range
- [x] Total number of transactions
- [x] Best-selling products
- [ ] Sales breakdown by payment method

---

## 👤 7. User Authentication & Roles

- [x] Login with username and password
- [x] Create Admin account
- [x] Create Cashier account
- [x] Role-based permissions
- [ ] Record which user completed each sale

---

# 🎯 MVP Workflow

The system should support the following end-to-end workflow:

- [ ] Add products
- [ ] Restock inventory
- [x] Search or scan products
- [x] Add products to cart
- [x] Accept payment
- [x] Generate receipt
- [x] Automatically update inventory
- [ ] View daily sales report
